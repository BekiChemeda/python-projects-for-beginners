# Grade Management System

A simple CLI-based system to manage Students, Courses, and Grades using JSON for data persistence.

## Features
- Add Students
- Add Courses
- Assign Grades to Students for specific Courses
- View Lists of Students and Courses
- View Student Grade Reports

## Files
- `main.py`: The entry point and CLI interface.
- `Student.py`: Manages student data.
- `Course.py`: Manages course data.
- `Grade.py`: Manages grade data.
- `utils.py`: Helper functions for logging and ID generation.
- `*.json`: Data storage files (created automatically).

## How to Run
Run the main script:
```bash
python main.py
```
Follow the on-screen menu instructions.
