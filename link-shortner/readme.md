## Link Shortener
A simple link shortener service built with Node.js and Express. This service allows users to create shortened URLs that redirect to the original long URLs.
### Features
- Create shortened URLs
- Redirect to original URLs
- Basic validation for URLs
### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bekichemeda/link-shortner.git
    cd link-shortner
    ```
2. Install dependencies:
3. Start the server:
   ```bash
   python main.py
    ```
